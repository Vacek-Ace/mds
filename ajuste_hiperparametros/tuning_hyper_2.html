Optimización de hiperparámetros: tune-sklearn¶

Tune-sklearn es un sustituto del mdulo de seleccin de modelos de Scikit-Learn (GridSearchCV, RandomizedSearchCV) con tcnicas de ajuste de hiperparmetros de ltima generacin.

pip install tune-sklearn ray[tune]

Scikit-Optimize: SkoptSearch¶

Scikit-Optimize, o skopt, es una biblioteca simple y eficiente para minimizar funciones de caja negra (muy) costosas y ruidosas. Implementa varios mtodos para la optimizacin secuencial basada en modelos. skopt pretende ser accesible y fcil de usar en muchos contextos.

La biblioteca est construida sobre NumPy, SciPy y Scikit-Learn.

pip install scikit-optimize

Optuna: Optuna¶

Optuna es un marco de software de optimizacin automtica de hiperparmetros, especialmente diseado para el aprendizaje automtico. Cuenta con una API de usuario de estilo imperativo, define-by-run. Gracias a su define-by-run API, el cdigo escrito con Optuna goza de una gran modularidad, y el usuario de Optuna puede construir dinmicamente los espacios de bsqueda para los hiperparmetros.

pip install optuna

Métricas de evaluación personalizadas¶

En ocasiones las mtricas dsiponibles para evaluar la calidad predictiva de un modelo de Machine Learning no son las adecuadas. Para ello podemos utilizar la funcin make_scorer y crear un scorer vlido de scikit-learn desde cualquier mtrica o funcin de perdida que decidamos utilizar, existente en scikit learn o creada por nosotros.
